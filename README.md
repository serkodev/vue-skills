# vue-skills

Agent Skills for Vue.js development.

> [!IMPORTANT]
> (WIP) This project is under development. The skills provided here are experimental and may change in the future. Use them with caution and at your own risk.

## Installation

```bash
npx skills add serkodev/vue-skills
```
